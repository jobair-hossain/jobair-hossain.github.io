My research focuses on developing intelligent, trustworthy, and scalable computational systems across healthcare, cybersecurity, and software engineering.

1. Multimodal Medical Data Analysis for Precision Healthcare

This research project integrates medical imaging, clinical data, and advanced machine learning techniques to improve disease prediction, tumor classification, and clinical decision support systems.

Sub-Projects

  • Medical Image Analysis with Deep Learning: Deep learning models for histopathological whole slide image (WSI) analysis.
  • Tumor Classification with Quantum Machine Learning (QML): Hybrid quantum-classical models for tumor mutation prediction and classification.
  • Reinforcement Learning-Based Fusion Models: Adaptive fusion of multiple models to improve prediction accuracy and robustness.
  • Generative AI for Clinical Decision Support: Evidence-grounded large language models for reliable medical interpretation.

2. Sentiment Analysis in Patient–Clinician Interactions

This project focuses on analyzing sentiment in patient-clinician communication using transformer-based deep learning models to improve healthcare experience and communication quality.

3. Quantum Machine Learning for Cybersecurity

This research explores the application of quantum machine learning techniques to address critical cybersecurity challenges.

Sub-Projects

  • Malware and Ransomware Analysis: Detection and classification using quantum neural networks and hybrid QML models.
  • Software Supply Chain Vulnerabilities: Detection of vulnerabilities in source code using quantum-enhanced learning approaches.
